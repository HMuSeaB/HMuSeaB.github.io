激活方法：

安装

PyCharm

，在安装过程中会出现一个激活窗口如下图

在此窗口中请选中 【Evaluate for free】然后再点击按钮【Evaluate】先以评估试用的方法进入PyCharm,然后再按下面的操作永久激活PyCharm

1、断开网络链接，下载PyCharm永久激活插件PyCharm永久激活插件2.3.5（提取码：9cqf ）

下载完成后不要解压压缩包

2、打开pycharm随便新建个项目进入主界面按下图演示操作

分别勾选【Logout when reset 】【Auto reset before per restart】

分别点击【Reload】【Reset】按钮

4、安装完成后恢复网络链接

永久激活插件的具体选项功能说明请看下面的介绍，

激活插件功能介绍

点击菜单【Help】->【Eval Reset】后会弹出插件主界面 ，插件主要功能介绍如下

●按钮【Reload】刷新界面上的显示信息。

●按钮【Reset】点击后会询问是否重置试用信息并重启IDE。选择Yes则执行重置操作并重启IDE生效，选择No则什么也不做。（此为手动重置方式）

●勾选项【Autoreset before per restart 】如果勾选了，则自勾选后每次重启/退出IDE时会自动重置试用信息，你无需做额外的事情。（此为自动重置方式）；如果不勾选的话每次PyCharm试用期到期后需要手动点击【Reset】按钮

●如果你的IDE已经过了试用打不开，可以运行这个临时重置脚本（注意选择对应系统版本），它可以让你暂时进入IDE进行上述操作

勾选【Auto reset before per restart 】并点击按钮【Reset】然后系统自动重启 PyCharm

重启完成后点击菜单【Help】->【About】查看是否续命成功如下图
